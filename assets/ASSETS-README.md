# untether — brand assets (canonical set)

Drop these into `~/repo/android-ble-to-ha/assets/`.

**Mark:** Bluetooth glyph (phosphor green `#39FF14`) broadcasting antenna bars +
nodes (electric cyan `#00E5FF`) — the Home Assistant antenna tell. Lit `FF` byte
and terminal caret also cyan. Background `#0A0E12`.

| File | Size | BG | Use |
|---|---|---|---|
| `untether-logo.png` | 2400×800 | dark | README header, docs/site banner, slide title |
| `untether-logo-transparent.png` | 2400×800 | transparent | overlay on any dark surface |
| `untether-logo.svg` | vector | dark | editable source for the lockup |
| `untether-mark.png` | 1024×1024 | dark | GitHub/org avatar, app tile |
| `untether-mark-transparent.png` | 1024×1024 | transparent | stickers, embroidery, tiles |
| `untether-mark.svg` | vector | transparent | editable source for the mark |
| `untether-badge.png` | 1024×1024 | transparent (die-cut hex) | DEF CON-style sticker |
| `untether-social.png` | 1280×640 | dark | GitHub repo social preview card |
| `untether-favicon.png` | 256×256 | transparent | favicon (glyph collapses cleanly) |

**Colors:** green `#39FF14` · cyan `#00E5FF` · mint (tagline) `#9FFFC8` · bg `#0A0E12`
**Type:** JetBrains Mono — wordmark 800, supporting text 500/700, all lowercase.

> The GitHub repo **social preview** (the card shown when the repo link is shared)
> is a web-only setting: upload `untether-social.png` under
> **repo → Settings → Social preview**. There's no CLI/API for it.
